Editor used: sublime text
Download link: https://www.sublimetext.com/3

You can use either .htm or .html as file extension. 
There is no difference between htm and html.

Open the html file with Google chrome to see the output.
Can use the Google developer tool to see the code in the browser.
Press (Ctrl + shift + I) to open it.
Select Source tab to view the whole code.
Select Element tab to inspect each element.
